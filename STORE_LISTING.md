## Long description

`dot lock pro` is a retro dot-matrix style lock screen for Windows with fullscreen display, multi-monitor coverage, and wake-from-sleep launch support.

It brings a distinctive dot-matrix visual style to your Windows desktop lock screen experience. The app displays time and date in a bold retro interface, covers multiple monitors in fullscreen, and is designed for clean always-on visual presentation.

The app is built for Windows desktop environments and works well for personal setups, kiosk-like displays, workspace ambience, and custom lock-style workflows.

It can also be configured to launch after Windows resumes from sleep, helping restore the lock screen experience automatically when your PC wakes.

Supports Windows 10 22H2 and later, including Windows 11.
